ROCK PAPER SCISSORS

Rock Paper Scissors is an easy game to learn because all a user needs to do is choose Rock, Paper or Scissors. The game will have a random choice and based off of the users choice, they can either win or loose. The code can be modified to create more options or modified to others liking!

userinput1 = input("Choose an item (Rock, Paper or Sissors):")

possibleactions = ("Rock","Paper","Sissors")

import random

comact = random.choice(possibleactions)

print(f"You chose", userinput1, "and computer chose", comact)

if userinput1 == comact:

print("Both players selected", comact, "it is a tie!")

elif userinput1 == "Rock":

if comact == "Sissors":

print("Rock defeats Sissors, you win!")

else:
