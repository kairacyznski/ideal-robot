MADLIBS

Madlibs is quite a simple game to catch onto and many people have played it before. The game is made with a story that is missing pieces and needs a user to insert them. The user does not know what the story is about so they will be inserting random words into the story which can make for a funny and random tale. The code can be edited or followed so that others can create their own fun Madlibs!
