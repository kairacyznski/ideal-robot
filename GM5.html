PRIME NUMBER FINDER

Prime Number Finder is alike to a calculator but it can find any prime number. If a user wants to find the 5th, 100th or any other prime numbers they just need to enter the prime number which they would like to find. The code can be changed to others liking or added to so that users can find other math equations.

count = 1

i = 3

list = []

userinput1 = int(input("Enter what prime number you want to find"))

while count != userinput1:

for k in range(2,i):

if i%k == 0:

break

else:

list.append(i)

count += 1

i += 2

print("The ", userinput1, " prime number is",list[len(list)-1])
