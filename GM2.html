GUESS THE NUMBER

Guess the Number is a fun and easy game which can be played by practically anyone. The user will attempt to discover a randomly generated number by making guesses. The game will tell you if you are to high or too low and will continue until you find the correct number. The code can be edited to make the range higher or lower so that others can create their own version of Guess the Number!

import random

print("Welcome to guess the number game!")

print("Try to guess a random number that is between 1-1000")

number = random.randint(1, 1000)

answer = -1

while answer != number:

answer_str = input("Enter your guess:")

answer = int(answer_str)

if answer < number:

print("Too low, try again.")

elif answer > number:

print("Too high, try again.")

print(f"Congratulations! You guessed the number {number} correctly!")
